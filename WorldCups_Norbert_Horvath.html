Probability and Statistics¶

I will work on the following dataset: https://www.kaggle.com/datasets/abecklas/fifa-world-cup

I will use the following .csv files:

  • WorldCupMatches.csv
  • WorldCups.csv

Let's dive deeper into the following hypotheses:

  1. $H_0:$ There is no significant difference in the average number of goals scored by the winning team between the group stage and knockout stage. $H_1:$ There is a significant difference in the average number of goals scored by the winning team between the group stage and knockout stage.
  2. $H_0:$ The proportion of draws in the knockout stage is the same as in the group stage.
  3. $H_0:$ The average number of goals scored by the host team in the World Cup is the same as the average number of goals scored by the other teams.

Where $H_0$ is the null hypothesis and $H_1$ is the alternative hypothesis. The alternative hypothesis is only written in the first point, because for the other points it can be deduced from the first example

import numpy as np
from scipy import stats

group_winner = np.array(winner_goals_group)
knockout_winner = np.array(winner_goals_knockout)
group_winner = group_winner.astype(int)
knockout_winner = knockout_winner.astype(int)

t_stat, p_value = stats.ttest_ind(group_winner, knockout_winner, equal_var=True)
print("t-statistic:", t_stat)
print("p-value:", p_value)

t-statistic: -0.4573739215546026
p-value: 0.6475528201300341

As we can see the $p = 0.64$ which is $p > 0.05$ meaning that the null hypothesis can not be rejected, meaning there is not a big enough difference to reject that teams which have won their matches scored differently based on the stage of the world cup

import collections

draws_in_group = collections.Counter()
draws_in_knockout = collections.Counter()
number_of_group_matches = 0
number_of_knockout_matches = 0

with open('WorldCupMatches.csv') as WorldCupMatches:
    csv_reader = csv.reader(WorldCupMatches, delimiter=',')
    
    next(csv_reader)
    
    for row in csv_reader:
        if (re.search("^Group", row[2])):
            number_of_group_matches += 1
            if (row[6] == row[7]):
                draws_in_group[row[0]] += 1
        else:
            if (row[1] != ''):
                number_of_knockout_matches += 1
            if (row[6] == row[7]):
                draws_in_knockout[row[0]] += 1

del draws_in_group[''], draws_in_knockout['']

for value in draws_in_group:
    if value not in draws_in_knockout:
        draws_in_knockout[value] = 0

These plots show us the number of draws in the group and knockout stages by year. For now we can't assume any major things about the hypothesis, but we can immediately see that from 1950 to 1978 there were no draws in the knockout stage. This implies that the hypothesis should be rejected, but let us test it.

To test the hypothesis we will use the z test. We can't use the student's t test, because the hypothesis is about proportions and not mean values.

In [ ]:
import math

group_draw_proportion = draws_in_group.total() / number_of_group_matches
knockout_draw_proportion = draws_in_knockout.total() / number_of_knockout_matches
se_diff_proportions = math.sqrt(group_draw_proportion * (1 - group_draw_proportion) / number_of_group_matches + knockout_draw_proportion * (1 - knockout_draw_proportion) / number_of_knockout_matches)
z_score = (knockout_draw_proportion - group_draw_proportion) / se_diff_proportions
p_value = stats.norm.sf(abs(z_score)) * 2

print("z-statistic:", z_score)
print("p-value:", p_value)

z-statistic: -3.5325175627624312
p-value: 0.00041162288377125816

As we can see the $p = 0.0004$ which is $p < 0.05$ meaning that the null hypothesis can be rejected, meaning there is a big enough difference between the number of draws in the group stage and the knockout stage. Which was exepcted.
